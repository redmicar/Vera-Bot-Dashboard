Introduction

Vera Bot is the perfect system bot to manage your server. The bot is community driven and can also be grown with your help.

About this Bot

The bot provides both SFW and NSFW features with NSFW features at it's core. The bot is hosted 24/7 and you can be reassured about it's uptime.

Technology

The bot is bulit with node.js using the discord.py library, same with the API the bot uses.

The bot uses it's own API to filter and provide high quality images to your server. With the help of our super fast and flexible API we are able to provide you the best and fastest features.

Getting Started

To get started with our bot is fairly easy and we have done our best to make it simpler for you.

Commands

Syntax:
  • < > : Required Argument
  • [ ] : Optional Argument
  • ( ) : Command Aliases

Moderation

  • Moderation
    • Description : You can moderate your server by our bot commands ex: ban / kick / mute / lock / purge message , etc.
    Usage : /ban < user and reason >
    Example : /ban
    Note : The Bot Must Have Adminstrator permission to access this commands.
  • AutoMod
    • Description : Let's you make system automod in your server and make it more secure from dangers link and 18+ attachments.
    Usage : automod debug < on or off >
    Example : /automod debug on | and use ex:/autolinks off
    Note : When You Turn on the debug mode make sure you select the automod action (autolinks - autoattachments).
  • Ticket
    • Description : Let You to open a advanced ticket in your specifec channel and your embed message with add category system
    Usage : /ticket setup
    Example : /ticket setup
    Aliases: search, id
    Note : By default doujins with tags which are against Discord TOS are not available through this command.
  • Giveaways
    • Description : Let You to start a giveaway with a specifec time and winners and prize.
    Usage : /giveaway start < setup message >
    Example : /giveaway start
    Note : you must setup the giveaway message after using /giveaway start.
  • Economy
    • Description : Now you can use our system economy to buy somethings and sell.
    Usage : /bank | /daily | /beg | /gamble< link >
    Aliases : imagesearch
